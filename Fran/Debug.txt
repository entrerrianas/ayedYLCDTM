 señala para abajo y seleccioná Run Configuration, te va a abrir la ventana Debug Configuration.
A la izquierda tenés como un panel ( Eclipse Application, Java Applet, Java Application, etc ), dale doble clik a Java Aplication, te va a abrir
 una pestaña nueva, apretá el botón search y seleccioná lo que vas a debagiar, apretá apply y después Debug, te va a saltar un cartel para que confirmes, 
mandale que sí.
Después te va a abrir el debug. Fijate en la solapa que tenes el codigo, hacé click DERECHO bien al borde y te abre un menu, fijate que dice breakpoint,
